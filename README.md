# 记忆岛

把取件码、取餐码、登机口等临时信息"钉"在通知栏上，像记忆一样常驻。

## 功能

- 短信自动识别取件码
- 通知栏常驻显示
- 定时提醒取快递
- 暗黑模式
- Android 16 灵动岛（ProgressStyle API）

## 支持的格式

**快递类**：菜鸟驿站、丰巢、申通、通用取件码  
**外卖类**：美团、饿了么取餐码  
**出行类**：登机口、座位号

## 技术栈

Flutter + SQLite + flutter_local_notifications + Provider
